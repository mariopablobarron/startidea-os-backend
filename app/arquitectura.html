Infra y software

Arquitectura en VPS + Coolify

Diseñada para ser simple de operar y facil de escalar por capas.

Diagrama logico

Usuario -> Web UI -> API/Automatizacion -> n8n Workflows -> SQL -> Observabilidad

El despliegue actual sirve frontend estatico por Nginx, mientras el backend y la capa de automatizacion se integran por endpoints internos.

Runtime

  • Docker como unidad de despliegue
  • Coolify para CI/CD y dominios
  • GitHub como fuente de versionado

Seguridad base

  • Variables sensibles fuera de Git
  • HTTPS con certificados automaticos
  • Registros de acceso centralizados

Escalamiento

  • Separar web, API y n8n en servicios
  • Agregar cola para procesos largos
  • Monitoreo de latencia y errores
