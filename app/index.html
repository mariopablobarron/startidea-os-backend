Bloques principales

Core LLM

Normaliza entrada, contexto y tono para que cada modulo de la plataforma opere con consistencia.

Router Inteligente

Deriva cada mensaje al subflujo correcto: idea, accion, estado, validacion o foco diario.

Operacion en VPS

Despliegue y entrega continua con GitHub + Coolify para publicar cambios por push.

Ciclo operativo

La plataforma sigue un ciclo corto de captura, priorizacion, accion y trazabilidad para sostener foco y ejecucion.

01 Captura idea o problema.
02 Clasifica tipo de solicitud y urgencia.
03 Genera tarea con objetivo y fecha.
04 Actualiza estado y mide progreso.

Estado del proyecto

Base dockerizada activa, workflows versionados y estructura lista para escalar frontend + backend.
